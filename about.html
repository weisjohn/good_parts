What?

These are snippets / notes from the book that I made / used to learn the techniques from John Resig and Bear Bibeault's Secrets of the JavaScript Ninja.

Why?

How?

I have a python script that builds a JSON object which maps those files that exist in the /chapters directory. I then use the jQuery.load() function to insert into the DOM and run the JS.
